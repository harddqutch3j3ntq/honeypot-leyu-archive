# honeypot-leyu-archive

## 项目简介

本仓库主要用于归档和发布若干独立 HTML 页面。这些页面内容各不相同，彼此之间没有关联，也不对应任何特定域名或网站。仓库的定位是一个通用的静态页面存档空间，方便后续查阅与分享。

## 目录说明

```
.
├── index.html          # 仓库首页（可选，用于展示页面索引）
├── pages/              # 存放所有 HTML 页面
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
├── assets/             # 页面所需的静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下放置所有 HTML 页面文件，文件名通常以数字或简短描述命名。
- `assets/` 目录用于存放页面引用的公共资源，如样式表、脚本和图片文件。

## 页面归档说明

本仓库中的每个 HTML 页面均为独立归档，它们可能来自不同来源或不同时期。页面内容以原始形式保存，未经修改或美化。仓库不保证页面的内容准确性、时效性或完整性，仅作为一份静态存档供参考。

- 页面命名尽量保持与原始一致，或采用便于识别的编号。
- 若页面包含外部资源（如远程图片、字体等），可能无法正常加载。
- 部分页面可能包含脚本或交互功能，但仓库不负责维护其运行环境。

## 维护说明

- 本仓库不定期更新，新增页面会以 Pull Request 或直接提交的方式加入。
- 仓库维护者会尽量保持目录结构清晰，但不对页面内容的合规性、合法性或适用性做任何承诺。
- 若您发现某个页面存在严重问题（如无法加载、内容错误等），欢迎提交 Issue 反馈。
- 仓库不接收外部投稿，所有页面均为内部归档。

## 使用方式

您可以直接通过 GitHub Pages 或其他静态托管服务访问 `pages/` 目录下的 HTML 文件。例如，在浏览器中打开 `https://<username>.github.io/honeypot-leyu-archive/pages/page-01.html`。

如需本地查看，请将整个仓库克隆到本地，然后用浏览器直接打开对应的 HTML 文件。

## 许可

本仓库包含的 HTML 页面内容可能来自不同原始出处。除非特别声明，仓库本身不持有这些内容的版权。若您认为某个页面侵犯了您的权益，请联系仓库维护者处理。
